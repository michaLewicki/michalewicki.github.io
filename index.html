Silnik Quizów
Silnik Quizów - pozwala na przeprowadzanie quizów dla kilku osób/grup w oparciu o własne pytania o różnym poziomie trudności. Uczestnicy wybierają jedną z czterech odpowiedzi (A, B, C, D). Wygrywa grupa, która zdobędzie największą ilość punktów.
Język: JavaScript, HTML, CSS
Szyfrowanie plików
Klasy Javy, służące do szyfrowania połączonego z kompresją wybranego pliku/folderu w oparciu o własny algorytm symetryczny (nazwany "Triple-XOR" :-)). Obok powszechnych rozwiązań zaimplementowano wiele własnych. Są ambitne plany rozbudowy, na razie hamowane przez chroniczny brak czasu.
Język: Java
